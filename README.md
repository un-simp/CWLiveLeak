## Config

Configs are where you would usually find them

```yaml
## Plugin GUID: CWLiveLeak

[General]

## What Icon is shown, available choices are: liveleak, hypercam, spookycam 
# Setting type: String
# Default value: liveleak
Icon = liveleak
```
not case sensitive
